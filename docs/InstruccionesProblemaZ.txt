--------------------------------------------------
Especificación del problema de ejemplo (ProblemaZ)
--------------------------------------------------

El Problema Z es un problema de juguete que pretende explicar cómo se espera que se presenten soluciones al proyecto.
La idea es mostrar con un problema sencillo:
- cómo se espera que se lean los datos
- cómo se espera que se escriban las respuestas

Problema Z:
Entrada: Cada linea del archivo contiene una lista de n números enteros, 0<n<=1000. El fin de la entrada se indica con una línea que contiene un 0
Salida : Por cada linea del archivo de entrada se espera una línea de la forma
np snp
donde np es el número de enteros pares leídos y snp es la suma de los números pares leídos.

Como en los problemas del proyecto, se esperan varios casos de prueba. El número de casos de prueba no se dice: se procesan casos de prueba hasta que se encuentre una linea que solo contiene el número 0

--------------------------------------
Instrucciones para compilar y ejecutar
--------------------------------------

Para compilar y ejecutar este programa seguir las siguientes instrucciones:

1. Abrir una consola.

2. Ir a la carpeta donde se encuentra el archivo ProblemaZ.java utilizando el comando cd:

cd C:\Usuarios\jorge\carpeta1\carpeta2\...\

3. Ejecutar el comando javac para compilar:

javac ProblemaZ.java

4. Ejecutar el programa:

java ProblemaZ

El programa va a quedarse esperando una lista de números de acuerdo con la
especificación del problema. Cada vez que se provea una lista y se presione
enter, debe aparecer la respuesta al problema en el formato esperado.

Para pruebas grandes es más cómodo leer un archivo y escribir las respuestas
en un archivo. Para esto se puede crear un archivo de texto con todos los casos
de prueba (por ejemplo Z.in) y usar la consola de la siguiente forma:

java ProblemaZ < Z.in > Z.out

El símbolo "<" redirige el contenido de "Z.in" a la entrada estandar.
El símbolo ">" redirige la salida estandar al archivo "Z.out"

---------------------
Trabajando en Eclipse
---------------------

Para usar los archivos compilados por eclipse, simplemente se debe ir 
a la carpeta donde están los archivos .class:

cd C:\Usuarios\jorge\carpeta1\carpeta2\...\ProyectoDAlgo\bin

y ejecutar directamente el programa. También se puede ejecutar java desde cualquier
parte del sistema de archivos utilizando la opción "-cp" para indicar donde están los archivos .class

java -cp "C:\Usuarios\jorge\carpeta1\carpeta2\...\ProyectoDAlgo\bin" ProblemaZ

Finalmente, utilizando rutas a los archivos se puede ejecutar el programa desde cualquier directorio:

java -cp "C:\Usuarios\jorge\carpeta1\carpeta2\...\ProyectoDAlgo\bin" ProblemaZ < C:\Usuarios\jorge\carpeta3\carpeta4\...\carpetaN\entrada.in > C:\Usuarios\jorge\carpeta10\carpeta11\...\carpetaM\salida.out
